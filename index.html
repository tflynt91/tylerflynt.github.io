Source Code

Created a web application for a car dealership using ASP.NET MVC. This utilizes a full-stack of software development skills. Developed the back end first, designing a fully normalized database, and then building repositories with a test-driven approach to make sure they were integrating with the database properly. Used Entity Framework to configure ASP.NET Identity in the data layer, and customize the user and user-roles. In the UI, the nav bar options change based on who is logged into the app, and what their role is. The search options make Ajax calls to an Inventory API, which in turn runs a repository method that creates custom SQL queries based on user input.

Source Code

Built a Web API with ASP.NET that provides basic CRUD functionality for a database of Dvd's. This project also includes a custom built REST client, and database. The API calls certain stored procedures in the database based on the appropriate CRUD method. It is configured to use CORS, and implements in-memory, Entity-Framework, and ADO.NET data retrieval via an interface. Dependency injection is used via factory class in order to implement a given interface. The database uses SQL security measures to grant access to different methods based on the user's privileges.

Developed a Battleship program for the console using C# and tested using NUnit. It implemets Object Oriented design patterns by abstracting the logic layer from the user interface. About

Born and raised in Buffalo, NY but have called many different places home during my career as a classical musician. I spent 10 years studying percussion and auditioning for orchestras accross the country. During my studies I've had many great experiences, from premiering operas to playing alongside The Boston Symphony Orchestra as a Tanglewood Fellow.
However, the transitory life of a musician has made me yearn for a more stable career in one place. Many of my collegues from over the years have transitioned into the world of tech, so I decided to give it a try, and ended up loving it! I find that the constant learning of new languages, and hours spent focusing on small details to be very similar to the hours spent learning and perfecting a piece of music.
The Microsoft technology stack is the environment where I spend most of my time, but I also enjoy working with front end frameworks such as React. When I'm not coding or drumming you can find me reading, cooking, or doing crossword puzzles.
